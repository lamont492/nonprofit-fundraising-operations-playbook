# Donor Lifecycle Process

## Step Based Process Map

### Step 1: Identify Prospect
Owner: Development Lead

Output:
- prospect added to pipeline

### Step 2: Qualify Prospect
Decision:
Does the prospect align with giving priorities?

If yes:
- assign owner
- move to cultivation

If no:
- document reason
- archive for future review

### Step 3: Cultivation
Owner: Relationship Owner

Actions:
- introduction
- meeting
- engagement tracking
- next step assignment

Output:
- ask readiness determined

### Step 4: Solicitation Planning
Owner: Development Lead

Actions:
- define ask amount
- define timing
- define executive support if needed

### Step 5: Ask Made
Decision:
Did donor commit?

If yes:
- create or update opportunity
- assign stewardship follow-up

If no:
- capture outcome
- define re-engagement timing

### Step 6: Gift Recording
Owner: CRM Admin and Finance Support

Actions:
- record gift or pledge
- link to campaign
- validate amount and payment details

### Step 7: Acknowledgement
Owner: Donor Relations

Output:
- thank you completed
- leadership touchpoint if required

### Step 8: Stewardship
Owner: Relationship Owner

Actions:
- impact update
- retention planning
- renewal or upgrade strategy

Final Output:
- retained donor relationship and accurate reporting history
