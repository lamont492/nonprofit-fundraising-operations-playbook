# Nonprofit Fundraising Operations Playbook

A practical public playbook for nonprofit fundraising operations, Salesforce NPSP planning, donor tracking workflows, sponsorship management, reporting design, and project governance.

## Purpose

This repository provides sample documentation, templates, and operating frameworks that help nonprofits improve fundraising execution, operational clarity, stakeholder alignment, and reporting discipline.

## What This Repository Covers

- Salesforce NPSP campaign structure guidance
- Donor tracking workflow documentation
- Fundraising operations templates
- Sponsorship strategy framework
- Reporting dashboard planning notes
- Project governance templates
- Implementation checklists
- Architecture and process notes

- Stand up a more structured fundraising operating model
